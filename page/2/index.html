三门问题，又被称为蒙提霍尔问题（Monty Hall Problem），它源自一个美国的电视游戏节目：Let’s Make a Deal。

这个问题是这样的：
参赛者会看见三扇关闭了的门，其中一扇的后面有一辆汽车，选中后面有车的那扇门就可以赢得该汽车，而另外两扇门后面则各藏有一只山羊。当参赛者选定了一扇门，但未去开启它的时候，节目主持人开启剩下两扇门的其中一扇，露出其中一只山羊。主持人其后会问参赛者要不要换另一扇仍然关上的门。问题来了：换另一扇门会否增加参赛者赢得汽车的机会率？

做自己的CEO笔记第四周

发表于 2016-10-24   |   分类于 笔记   |  

经典经济学有两个假设：
1.信息总体是对称的，你知道的我也知道。
2.人总体是理性的，都会做对自己做有利的事。
但是行为经济学告诉我们，人在很多的情况下，并不那么理性。

结果偏见

  • 我们看到一个人取得了成功，就立刻认为他过去所有的行为都那么的有道理。可成功者认为的那个经验，可能是他获得更大成功的绊脚石。
    不能不管黑猫白猫，抓住老鼠的就是好猫，要分得清哪些是实力，哪些是运气。
  • 如何避免结果偏见？
    1.在归纳法上加上演绎法。从结果推到原因之后，要再看原因是否能真的推出正确的结果。
    2.做前3问。
    这个结果真的有一个人为可控的因素存在么？
    这个人，真的知道那个人为可控的原因是什么么？
    他引以为豪的，是不是恰恰是他的瑕疵呢？
  • 结果偏见就是外人看成功者羡慕，看失败者惋惜。
  • 归纳法让我们像一个律师一样，看到了结果，到事件中寻求支撑成功的证据。演绎法让我们像一个科学家一样，有了证据，再去推结论。如果只有归纳没有演绎，对总结出结论就会像情人眼里出西施，怎么看都对。
  • 高概率的成功方法比某一次成功的结果更重要。

首先跟大家说清楚今天的主题：如何打造且不断升级我们的操作系统，我们说的是阅读操作系统。大家一定要清楚我们讨论的是知识性阅读，不是娱乐性阅读，读诗歌、小说、散文虽然同样需要我们今天提到的这些技能，但我们今天的讨论聚焦在如何通过阅读获得知识进而改变生活。其次，我用了操作系统这个词，其实就是一个类比：我们可以把我们自己想象成一台电脑，至于这台电脑是古董级还是超级电脑，是靠我们自己的，或者说从一开始就是由我们自己决定的。

—输入（键盘、鼠标、麦克风、摄像头、扫描仪）

李笑来老师知乎Live《我的读书经验》全文

发表于 2016-10-23   |   分类于 笔记   |  

1.我的读书路径与境界更新

读书这件事在最初的时候很难做到目标明确，而且，一上来就目标明确的话，很可能会造成另外一个严重后果，即：失去获得意外好运的机会。所以，在第一个阶段里，我觉得甚至直接就应该漫无目的的读书，能读多少就读多少，能读到什么就读什么。不用挑，因为挑书也是一种逐步积累的能力。回顾我的读书经历，大约在28岁之前，我读书是非常杂的，连那时候男生都不屑于读的言情小说我都可以读的津津有味，现在回头看，这种饥不择食有很大的好处，随便罗列也起码有这么3个：

1.阅读变成了吃饭一样的需求，不读脑子就饿……

2.经常有各种意外的好运，从莫名其妙的地方读到了原本可能不可能获得的知识。

3.一不小心就积累了足够的里程—— 你要知道，驾驶经验远比驾驶技巧重要得多……

我有一个当图书馆馆长的妈妈，所以从小就懂一些基本的选书原则，因为我早就公开写过了，所以在这里我就不重复了，大家只要关注我的微信公众号【学习学习再学习】，在后台回复“选书”两个字，就可以看到我那篇文章：《选书的基本原则》。

到了第二个阶段，你就不得不挑书了，因为挑书的能力是在大量的阅读过程当中必然逐步形成，我从来都不向别人索要书单，因为最初就觉得书单是没有什么用的，那肯定是别人的书单，那肯定不是我的书单。后来，就庆幸了，因为从来不向别人索要书单，于是，思考久了观察多了，就明白一个特别简单的道理：向别人索要书单是使自己的挑书能力逐步退化直至为零的最有效方法。

公众号青衫磊落对于剧本的看法

发表于 2016-10-17   |   分类于 笔记   |  
  • 剧本是一剧之本，所以编辑的功力很重要，成为知名编剧需要有才华以及好运气。

剧本招数(剧情剧drama)

1、从草根人物写起。

  • a让普通观众可以把自己放在角色身上，感同身受，有亲近感。平凡的人总有不平凡的故事，反差明显，吸引眼球。
  • b着重描写正值中年的市井小人物，他们经常处在挣扎着找回自我的阶段，苦守如日中天的曾经，又深知难逃日薄西山的宿命。
  • c让主角疾病缠身，或有着惨痛的回忆。草根人物也会做出出格的事情，使主角患上心理疾病或常见身体疾病会让观众心生同情，但是一定要解释清楚获得这些疾病的原因，以理服人。
  • d让他有一技之长，而且天赋秉异，让观众佩服他，这样观众会宽恕他的过失。他的专长也必定是剧集的重心。但是他虽才华横溢，也不是所向披靡，必须有一个旗鼓相当的对手让他有危机感，然后使他为了达到自己的目的做出极端的事情，增强戏剧性。
  • e让他的事业成为美国梦的缩影。让观众对普通职业产生共鸣，与贪得无厌的人做对比。
  • f给他一个不可告人的秘密。容易描绘出一个与自我内心斗争的戏码。秘密也会推动情节发展，无论秘密是什么，都会为现实的家庭生活，婚姻问题，社会地位及信任问题提供切入点。
  • g用女性当主角。扮演咄咄逼人神经兮兮的小人物。

美剧American Crime Story感想

发表于 2016-10-17   |   分类于 笔记   |  

  • 刷了整季的American Crime Story，感受到了美国司法的严谨，客观，对美国司法肃然起敬。

  • 这一季讲述的oj辛普森杀人案完美地体现了疑罪从无的原则，即不能证明被告人有罪又不能证明被告人无罪的情况下，推定被告人无罪。

  • 又清晰的展现了陪审团制度的伟大，所有公民在称为陪审团之后被临时赋予公职，所有人的目标都是判断这些证据足不足够将他判罪，类似于十二公民中的讨论过程。

  • 季中检方的证人选择失误，警方处理程序失误，以及辩方对证人的心理威慑和对陪审团的疑点分析，外加种族问题的背景渲染，让最终的结果称为无罪释放，即使真相可能就是辛普森杀妻，但是依据法庭来判决的最终结果就是无罪。

  • 另外学到的法律零散知识包括，第五修正案法，意思就是如果你认为我有罪，那么必须你拿出证据来证明我有罪，而不是我拿出证据来证明我无罪；

  • 证人发誓不能说谎；
  • 一案不再审原则；
  • 刑事诉讼与民事诉讼对证据要求的不同，刑事诉讼由于要对被告人处限制人身自由甚至剥夺生命权的判决，必须让证据具有排他性，绝对性才能作出判决，否则只能疑罪从无。而民事诉讼则由于判决一般只涉及金钱赔偿，可以根据常理进行推断。因此，民事判决是否就是客观事实不能绝对化；
  • 受英国普通法中“品格证据”传统的影响，美国联邦和加州的证据法和判例都规定，如果出庭证人的品格被证明有缺陷，则证人呈庭的某些证词就不具有法律效力。

由之微观经济学入门笔记

发表于 2016-10-17   |   分类于 笔记   |  

1、经济。

  • 划算的意思。满足一定需求的情况下成本最小，或者一定成本之下总效用最大。经济学核心就是优化的思想，换句话说就是权衡。

2、稀缺。

  • 1).资源在物理上是有限的，但人的欲望是无穷的，一个人也会产生稀缺。2).差异性带来稀缺，好的总是少的。从个人来讲，利用资源让幸福最大化。从社会来讲，资源配置让社会福利最大化。体现了经济学背后优化的思想。

3、竞争。

  • 一项资源到底归谁的事情。两个人以上必然产生竞争。解决竞争的办法是找到一种配置稀缺资源的机制。

4、机制。

  • 时间机制，先到先得。运气机制，抽签中奖。武力机制。身强力壮可以占领资源。权力和等级机制，皇权。价格机制。
  • 经济学以价格机制为核心研究，可迁移性强。价格机制是闭环的，利己以利他为前提。本质是促成了商品或服务的交易。给人最合适的激励，给人减少的交易成本。
  • 在价格机制下做决策时权衡成本和收益。
