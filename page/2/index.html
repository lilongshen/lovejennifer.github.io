  • 在市场条件下，想要实现有限的交易，交易双方掌握的信息必须对称，如果不对称，掌握信息越充分的人获得利益越大。

做自己的CEO笔记第二周

1、流量成本

  • 零售的核心就是流量成本。我们每获得一个潜在客户，要为此付出的基本价格。企业盈利的关键在于寻找低成本的流量。
  • 流量成本＝成本/潜在客户数。销售=流量×转化率×客单价。
  • 互联网电商的崛起是某个时段的产物，是由于互联网人数消费者越来越多，但商户非常少，所以电商流量成本极低，人们搜寻产品购买，只能搜到几家店铺，所以每家店铺客流量极大。但是现在同质商品增多，开始做广告，竞价排名，竞争得到更多的流量，直到广告价格和成本与利润持平为止。所以电商不是什么先进模式，只是恰好处在互联网流量成本低的时刻。先进模式是各种流量来源中流量成本最低的模式。关键在于找到最低的流量成本渠道。比如社群，比如自媒体，比如直播，比如依靠口碑宣传推荐，比如吸引老顾客重复购买，比如到租金低的地方开店。

  • 表达式与语句。表达式就是一个一个值，类似于英文中的单词，语句是为了完成某种任务的操作，以分号结尾，类似于英文中的句子。

  • 关键字和保留字。它们不能做变量名，完整版如下：
    “break case catch class const continue debugger
    default delete do else enum export extends false

    protected public return static super switch this
    throw true try typeof var void while with yield”
  • 环境。变量和它们的值所在的空间被称为环境。当程序启动，环境不是空的。它包括语言规范的变量，更多时候是与操作系统交互的变量，就浏览器而言，包括检测到的当前网页的变量和函数，鼠标和键盘的输入。
