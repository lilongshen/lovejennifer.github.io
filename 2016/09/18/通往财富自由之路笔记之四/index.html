通往财富自由之路笔记之四

发表于 2016-09-18   |   分类于 笔记   |     |   本文总阅读量 次

本次学到了两个概念，资本与落后。资本不仅仅是资金，落后不仅仅是你所想的那么轻松。

你到底有没有资本？

什么是资本？

  • 房子是砖头建造的，一堆砖头堆在一起，算不上房子。资本是钱组成的，一堆钱放在一起，算不上资本。资本的概念在以下三点。
  • 资金的金额大小。100元不算资本，100万放在什么领域才算资本。
  • 使用时间。如果拥有10亿只有1天，无法增长，无法算资本。时间越长越有效。
  • 背后的智慧。

对资本认识的突破点。

  • 突破1：看盈亏比例，相对值。不看盈亏金额，绝对值。注重资金效率。本金大小不重要，重要的是盈亏比例。
  • 突破2：将一部分资产判无期徒刑，坚持每月投入，分出来哪部分是存钱，哪部分是可用资金。长期思考未来的能力。
  • 突破3：一切经验都是实战出来的，没有什么东西可以取代经验。

审视自己的问题。

  • 问题1：曾经有哪些投资机会做错？

    我收藏夹里的火币网缓存还是2015年3月的时候1384.44元人民币／个，现在已经是4147.30元／个了，当时如果咬咬牙，投资几个的话，现在也不会只靠工资和存钱活着了。
  • 问题2：是否有过将一部分资金判无期徒刑的经历，预期是多少？

    曾经月入500元的时候，存了1000元定期存款，当时看过一些理财的书籍，其实就已经了解过每月固定存钱，保持不动，甚至转到一张存折或从来不用的卡上，可能都忘记自己有过那一笔钱。之后也坚持了两个月每个月存500元，但是由于自己的购买欲和收入的不稳定，无法保持存款，同时也是自己内心不愿意让自己的资金固定不动，心理焦急地等待那1年定期赶紧到期取出来，好像没有那1000元就活不下去一样，那种心情现在想起来都觉得很傻很天真。未来我预期每月继续保持工资或每月固定收入的15%判无期徒刑，放在支付宝不再动用，现在就是300元，将来争取到每月2000元的时候，开始投资股票和基金，寻找盈亏比例高的股票。
  • 问题3：哪些是你必须做，哪些是你绝对不能做？

    花掉已经判无期徒刑的钱，绝对不能做。开始收集盈亏比例高的股票信息，必须做。区分哪些是旅行娱乐的费用，哪些是固定不动的费用，哪些是基本生活的费用，必须做。如果比特币再次跌到1500位置的话，必须买。超过自身能力使用信用卡借款消费，绝对不能做。拿生活必需费用来投资，绝对不能做。

如何走上资本增值之路。

  • 重温复利曲线：每天进步一点点加长时间的坚持，最终能达到质的飞跃。不要关注别人已经走了多远，重要的是自己是否已经迈出了第一步。
  • 只有投资才能走上财务自由之路。广义的投资包括每天如何分配自己的时间才能获得最多的回报。有限的注意力投入到哪些领域才能得到有效的产出。时间的损失给人的刺激不大，但是金钱的损失给人刺激很明显，本质是一样的。学会如何投资，就是学会如何将注意力投资，如何学会资本背后的智慧。
  • 切记不要浪费自己宝贵的注意力，比如不要一直盯着大盘，不要迷信专家，所有投资成功都是舍我其谁的独立思考的结果。
  • 用一个正确概念取代错误概念，用暂时未成功，代替失败。用一个新习惯代替旧习惯，注意自己的成长与否，不注意金钱增减，转变注意力。用一个新想法取代旧想法，把判无期徒刑的钱当作丢了的钱。

落后

落后的概念是什么？

  • 及格没有任何作用，二八定律无所不在，这个定律是1897年意大利经济学家帕累托研究英国人财富和收益模式得出。20%全是落后，甚至发展到今天，1%之后都是落后。

改变落后的方法论。

  • 所有的国外高学历的人的生活真真切切地告诉自己，知识改变命运。很多知识是传统教育方式无法获得的，目前所有人对知识的渴求是强烈的，所有人都不想落后于时代，那么这就是创业的方向，提升别人，满足求知欲。
  • 多了解心理学的偏差，是避免掉进思维缺陷中的陷阱，理清概念，促使自己不断进步。

    乌比冈湖效应（Lake Wobegon Effect），也称沃博艮湖效应，意思是高估自己的实际水平。社会心理学借用这一词，指人的一种总觉得什么都高出平均水平的心理倾向，即给自己的许多方面打分高过实际水平。用另一种通俗的说法，就是自我拉抬偏差(self-enhancing bias ).

    杜宁-克鲁格效应，是“越差越牛屄，越强越谦虚”的现象。倾向于高估自己的能力水平；无法认知他人真正的能力；无法认知且正视自身的不足，及其不足之极端程度；如果他们能够经过恰当训练大幅度提高能力水平，他们最终会认知到且能承认他们之前的无能程度。-未来的目标：1、早晚有一天要不为生活出卖自己的时间。2、早晚有一天要成为世界中的前20%，甚至前1%。
  • 长期持续就是最有用的诀窍，很多人在路上都逐渐放弃了。长期写作就是一个方向，因为持续写作就是在积累获得被动收入技能的练习。用写书的态度和功夫写文章。

实践如何进步

  • 学英语的最大问题就是对自己的要求太低了，不断提升自己的标准，要不断追求。
  • 所有差不多的用词其实都差很多。很多事情都不是线性发展的，由60到80分很容易，但是由80到90分非常困难。不要让自己感觉差不多。
  • 其实靠知识获利，往往是做别人不愿做的事，别人懒得做的事，或者别人没法做的事。正如翻译知识性文章，很多人觉得英文困难，不会英文，或者不愿意阅读，或者懒得翻译。如果自己可以持续不间断的翻译文章，直到日后可以选择性模仿，然后写出独具自己风格的文章，这就是知识获利的途径。笑来可能是这样，比特币的恭亲王可能是这样，公众号的小肖写金融可能是这样，公众号媒体知识传播的新闻实验室也是这样。
