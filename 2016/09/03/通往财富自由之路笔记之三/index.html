通往财富自由之路笔记之三

发表于 2016-09-03   |   分类于 笔记   |     |   本文总阅读量 次

人生最重要的枷锁是什么？

  • 不愿放弃部分安全感，寻求稳定，不愿改变，想维持现状，缺乏深入长期的思考就是枷锁。勇于放弃部分安全感的人不满足现状，在创新，用脑力来推动社会进步。历史上尝尝聚众屠杀这些推动社会进步的人，本质上这是不愿改变、不愿放弃部分安全感的人对改变的深深恐惧。有时候狗咬人不是因为它凶狠，是因为他恐惧。改变自己的命运就是要创新，放弃部分安全感，提升能力。

放弃部分安全感能得到什么？

  • 抛弃每个话题都有参与感的安全感，注意力会更集中，你会更珍惜自己的注意力。抛弃追求稳定的安全感，眼界会更宽广，你会得到更多的机会。减少看综艺节目和体育节目的时间，专注在学习和成长上。积累专业知识，同时跨界产生连接。

为什么不愿意放弃部分安全感？

  • 比较别人眼中的好形象带来的满足感和自身进步带来的满足感哪个更多。人装可以骗别人，但是不愿花时间进步，是没法骗自己的。要体会进步带来的快感。

例如读书不要求数量来满足安全感

  • 读书最重要的是学以致用，应用读到的知识改善自己的生活，不要追求谈资，不要追求数量。

如何面对他人的反对看法？

  • 不要在意他人的反对，要把别人的鼓励和支持当作赠送，不要寻求认同和认可。就像你确信你知道下周开奖的彩票号码，你不会寻求他人知道和认同，等着中奖就好了，结果会证明一切。因为你认可你的判断力，你知道你拥有和追求的东西是有价值的。如果你做一件事情非要让他人认可，那就是你不相信你正在做的事情是未来需要的事情，不认为是有价值的事情。无论他人怎么看，开始行动就好了。不要停留在口头上，要落实在行动中；不要只做一点点，要持续的积累。大多数人都会做那么点，重要的是能不能持续积累，让自己的优势显现出来，厚积薄发啊。

不要相信捷径

  • 不要相信有捷径，没有万能公式，所有的路都是自己走出来的。

如何活在未来？

  • 要对未来的预测，笃信预测，并开始运用实践预测，随着时间积累，你将引领潮流，活在未来。对未来的预测举例：长期来看，脑力的产出率一定比体力的产出率高出很多。知识产生更大的暴利。未来是程序员的世界，所有事情可以用来编程，人将活在代码当中，所以现在开始学习，开始创造，这就是活在未来。知道道理，笃信道理，执行这些道理，不断地习得，不断地改进，不断地积累，就是活在未来。

请乐于分享

  • 所有的道理都可以言无不尽告诉所有人，因为自己在不断地学习，不断地追求新的道理，更重要的是自己笃信这些道理，自己在不断践行，不断改进，不断积累。所以请尽情分享，不要犹豫，帮助他人是快乐的，也是理清思路，加速成长的途径。

给自己的任务

  • 除了记录每天的注意力分布和遇到事情的感想之外，还要在每天的总结中归纳周围发生的事情的规律和趋势，看看未来可能发生什么。
  • 想成为什么样的人，就按照那样的标准来要求自己，不要等待时机成熟，开始用github，开始用nodejs，开始学习最前沿的前端知识，就是活在未来，就能尽快成为专业人士。每天花些时间，看专业内最新的报告，预测专业内的未来，深入思考，大胆求证，积累证据，分享给他人，听取反馈，思考。将自己正在学习的领域用专题或者书稿的形式来迫使自己系统的梳理。用写书的方式来写博客。

清晰思考，踏实实践

  • 优质的思考就是活在未来必备的条件，严谨的逻辑能帮助人更好的活在未来。尊重事实，相信逻辑，无视非议，这都是活在未来的人应有的品质。大部分人停留在想一想的阶段，就是没有笃信自己的判断。笃信的根基不是盲目的自信，来自于扎实的推理和不断的拷问，用足够的数据支持观点，不轻易下结论。真正开始做功课就会发现需要补充的资料还有很多，能让自己笃信的背景信息还不充分。这个过程中就会摆脱盲从，认真对待每一个观点。在行动中验证，不断修正自己的想法，最后预测不正确，但是这个过程会得到很多成长。

如何提高遇到贵人的可能性

  • 乐观的人更容易成为他人的贵人。积极的生活态度是有极强的吸引力的，这种人让人觉得人生充满无限美好的可能。
  • 贵人更容易遇到贵人。自己成为贵人是最终目的，不是遇到贵人的手段。家有梧桐树，自有凤凰来。只有到了那个层次，才会有相应的圈子，而不是倒过来。
  • 能帮助他人进步的才是真正的贵人。 帮助他人解决关键问题的人是真贵人。
  • 优秀的人，值得尊重的人更容易得到帮助。 其实与遇到贵人先成为贵人相同。
  • 乐于分享的人更容易获得帮助。 贵人的精神总是相同的，就像黑客的调调总是想通的一样。
  • 不给他人制造负担的人更容易获得帮助。 不要让别人心理难受，不要用自己的懒惰和贪小便宜心理恶心别人。
  • 不耻于求助的人更容易获得帮助。 其实重要的是减少犹豫，大胆说出“您好，请问”那一步，面子没那么重要。
  • 求助的时候不宜只用金钱作为回报，帮助他人的时候不宜索要金钱回报。 即时性回报例如金钱，不利于长期的发展，阻碍了未来交流和交友的可能性，个人认为知识变现不应该体现在问答上，理由如上。还是体现在付费内容上是最优选择。
  • 贵人不一定是牛人，贵人已和为贵，帮助他人成长，和身边的人一起成为贵人，增加贵人的密度，让生活中充满贵人，让生活更美好。此外，普通人在自己的能力范围内也可以帮助别人，每个人在不同领域都有自己的比较优势。
  • 很多时候一个人的成功，源自于大量的人希望看到他成功。不谈什么势能，气场，人和之类虚头巴脑的东西，众望所归带来的就是正面的心态，友好的环境以及潜在的受助可能。
  • 正在做正确事情的人容易得到贵人相助。贵人都是希望世界变得更加美好的，你做的事是使世界变好一点点，那自然得道者多助。
  • 活在未来的人容易得到贵人相助。

大多数人的缺陷

  • 绝大多数人无法等到一个时间段之后看结果，不愿意积累，不愿意耐心等待。大多数人都没有思考长远，耐心验证，小心总结提炼的能力。

求助的本质

  • 求助是一种交易。贵人愿意帮助你，是因为他从你身上看到了价值。要么就是自我实现的感觉，被需要也是一种幸福，要么就是看到了未来某种可能性，尽管并不确定。被需要也是一种幸福。所有人都生活在价值交换，价值集群之中。所有人都储备人际价值，到人际高的地方扎堆。求助是展示自我价值的艺术。

花精力在如何把问题表达清楚

  • 花时间和心思在把自己的问题和想要得到的帮助说清楚，讲明白，而不是花精力在如何委婉的表达和心理犹豫不好意思上。之前看到讲如何提问的文章，必须要在已经用搜索引擎查找后，在自己尝试过几种解决方案后，把自己遇到问题的前因后果，上下文理清楚后，这些都没有得到答案后的问题再询问他人，要表明你所做过的努力。另外要注重细节，很多时候出问题是因为细节遗漏，或者对细节概念不清。编程问题的话就更加严格了，要说明电脑配置，环境配置，谷歌后相关解答，自己问题详情，把代码上传到github等等。这个过程是对自己负责，是培养解决问题能力的方法，同样也是对求助者的智力尊重，让对方感受到你为问题付出的努力，让对方提高帮助他人所收获的满足感。毕竟随便就能搜出来的答案，不做努力查找，不尝试自己解决就问问题，这种人很容易被人鄙视，以自己的懒惰和贪小便宜心态来侵占他人的时间是可耻的，伸手党总是不受人待见，当然更不可能遇到贵人了。

如何面对贵人的帮助？

  • 对人情债的愧疚感可以尽量消除，把自己当成信用卡，给自己一定的还款期限，培养紧张感，告诉自己一段时间后就有能力还。告诉自己只是暂时能力不足，暂时资源有限，暂时成长缓慢。面对目前自己处于弱势地位，经常接受人帮助，想要最快报答对方的方式是飞速成长，尽快能帮助对方，而不是想法设法来维持关系。
