互联网是什么？

  • 互联网是巨大量的独立网络组合，是完全分散的，个体和个体可以任意相连。信息共享是巨大的能量，你有什么有价值的东西和别人分享，别人反之。反正你逃离不了互联网，那么为什么不了解它并使用它。

互联网由什么组成？

  • 互联网像邮筒，但是传递的不是实物，而是二进制信息binary information，信息由比特bits组成，bit可以被描述成一对相反的物件，开关，对错，10。8 bits = 1 byte，1000 bytes = 1 kilo byte，1000 kilo bytes = 1 mega byte。

信息传播？

  • 最初的信息传递非常慢，错误百出，而且太依赖人类。后来开始利用机器，从telegraph电报、braille盲文板、semaphore信号机到今天的electricity，light和radio。一条电缆绑两个灯泡就可以模拟电传播bits，1表示on，0表示off，发送5个0，则通过发送方和记录方约定，记录时间1秒1bit来传输。

  • 当然我们希望单位传输时间比1秒再快一些，所以我们需要增加bandwidth带宽，指一个设备的最大传输能力，用bitrate比特率测量。比特率指给定时间内实际发送的比特数量，通常以秒来计算。不同的速度测量叫做latency延迟，是指一个bit从发送者到接收者所用的时间。8 million bits = 1 megabyte，所以bitrate = 8 Mbps。人类没法这么快，但是机器可以。

做自己的CEO笔记第八周

企业能量模型

  • 企业经营，就是一个能量生产、能量转化的过程。产品生产势能，营销和渠道，把势能转化为动能。
  • 想象一下，一个人正在推巨石上山。做产品，就是把这块千钧之石推上万仞之巅，获得尽可能大的势能，然后在最高点一把推下去，用营销和渠道减小阻力，把势能转化为最大的动能，获得尽可能深远的用户覆盖。
  • 盈利困难的根本问题是产品势能不足。然后产品不足，营销补；营销不足，渠道补，最后只好陪客户喝酒、吃饭、搞关系，但还是卖不出去。喝酒、吃饭、搞关系都卖不出去的东西，互联网，也帮不上什么大忙。
  • 如何把企业做好
    • 第一，把产品这块巨石，推得越高越好。产品的创意、独特性、品质，或者说它积蓄的势能，决定了它最高可以达到的销售量级。没有势能的产品，是卖不出去的。就算能卖几件，其实卖的不是产品，是人情。172人笔记
    • 第二，然后你在山顶一推，巨石开始下滑，势能转化为动能。营销，就是用来减小下滑的阻力。广告、公关、线下活动、热点营销、加入行业协会，去拿各种奖项，都是为了提高客户对你的优先选择概率。
    • 第三，巨石开始水平滚动。这时，你用渠道继续减小阻力，通过大量铺设渠道的方式，线上、线下，电话、互联网、上门推销，甚至到达田间地头的方式，让消费者一旦有了购买欲望，你的商品就触手可及。

精英日课笔记之一

  1. 人脑每时每刻处理几万个进把程，每一个决定都是大脑中各种进程竞争的结果。人脑中有额内叶区，负责所有的进程汇总，最终互相竞争，权衡，得出结论。跟计算机比我们反应慢，动作慢，是因为我们时刻都在做各种临时的决定。
  2. 冷认知和热认知。冷认知是有意识的考虑新情况，临时做判断。热认知是大脑中已经安装好的程序自动运行。人脑依据习惯自动运行的时候像电脑处理。有一定的好习惯，熟练做某些事情是必要的。但是大脑不能变成只安装一个程序的计算机。有时会遇到这样的人，表面上看起来反应特别快，实际上他遇到任何情况都是那几招，三板斧用完就不会了。完全可以预测他的态度和行为。
    所以遇事多几个心眼，多从不同角度看问题，尤其是不熟悉的局面，具体问题具体分析。
  3. 判断是不是好的忘我体验，标准是做完一件事以后感到满足还是空虚。看到这句话第一反就是这两个例子。一个例子是看完一周一更的美剧或者综艺节目之后很失落，很空虚，总是觉得还想再看下一集或是类似的节目。但是自己按照教程一步一步做出的博客就感到非常满足。另一个例子是看色情片自慰之后非常空虚，情绪低落。但是和心爱的另一半云雨之后，感到非常满足，带着微笑相拥而睡。同样时间不知不觉的过去，但是结果会给你反馈，告诉你什么是正确而有益的事情。

做自己的CEO笔记第七周

风险投资

  • 风险投资，是通过买走创业的“失败风险”，让创业者拥有往成功目标一路狂奔的金融工具。
  • 早期我们通过讲一个故事让投资人相信我们未来的企业是有想象空间的，所以越早期天使投资人其实往往是不理性的，这个阶段的投资人大多会投一些跟自己阅历性格气场相符的创业者。而到了VC阶段需要故事+数据，不仅有个动听眼前一亮的故事还需要数据作为故事支撑的载体才足够打动VC。而到了后期PE因为比较接近二级市场，所以故事能起到的作用并不会很大，所以主要看数据看营收或者有上市对赌条款，毕竟做PE投资基本都是赚个中介费，临门一脚上市收钱。这就是创业的3个阶段，天使看故事，VC看故事+数据，PE看营收。
  • 正是因为其本质仍然是个概率游戏，风险投资当然有大量的失败案例。目前国内国企的急功近利和民企的小富即安，相对于风险投资，他们更愿意做拷贝性投资，并以低成本和环境代价来参与市场竞争。一个优秀的风险投资人需要对产业有全面深刻的了解和预判，需要在很短时间内对一个创业项目做出准确的判断，需要了解人性的贪婪与勇敢，需要敢于投入、押注和妥协，更要有资源配置的能力和健全的退出安排。作为创业者，留好足够的资金给爸妈妻儿，然后All in。记得徐小平说过，风险投资第一投人，第二投人，第三还是投人。所以说，修炼自己的能力、格局和三观才是最重要的。赚钱才是核心，大讲情怀的同时一定自己要清楚什么是第一位的。融资之后如何与合伙人、投资人一起求同存异才是最重要的！

逻辑学习

Knowledge知识

尝试解释一下知识：

  1. 知识是被验证为真实的信念。
  2. 关于真实。相信一件事并且这件事是真的并不是知识，因为缺乏验证。而认识论所解决的问题就是怎样恰当地证实真实的相信。如何验证事实就是各种主义的争论。
  3. 知识论的核心争论在于是否承认先验存在，选择承认就是以逻辑学和数学作为证据。尽管直觉上很难承认，因为直觉上看来所有的知识都源于经验，由观察经验总结得来，属于后验。
  4. 关于相信有两个误区：没有充分证据就相信事情真实性是非理性的恐惧；认为一个人想了解一件事就必须相信它是真的，这是非理性的判断。知识与信仰和观点不同，某人声称他相信一件事是事实，当然他有可能犯错误，因为这件事不一定是真的。这就是信仰和观点的特点。但是知识不同，知识一定是真的，经过验证的，且你相信的。

  5. 区分知道与能不能做。一个人只是知道如此(knowing-that)但是没有做到知道如何(knowing-how)，那么就是解说员与运动员的区别。换句话说就是解说员有命题性知识，却没有过程性知识；也就是知道理论，但不会做。

奥派经济学学习

Austrian School

  • 只有在逻辑上出自于人类行为原则的经济理论才是真实的。
  • 学派的成员将经济学作为一种先验的（a priori）系统来研究，如同逻辑学和数学，而非作为如地质学一般的实证科学。
  • 先验意味着仅凭推理得到的知识（先于经验观察），而不受直接或间接经验（这里经验通常指通过感官对于世界的观察）的影响。后验指其他种类的知识，也就是知识的得来和证实需要借助经验（经验观察之后），也被称作经验性知识。)
  • 他们认为价值是由个人的主观偏好所决定的。强调经济的决定是由个人而非集体所达成的，同时也包括边际主义的理论，以此来比较成本和利润的增长改变。
  • 政策要求政府减少管制、保护私人财产、并捍卫个人自由。

宏观经济学

节约悖论。

对国家来说，越节约越穷，越消费越富有。

  • GDP。GDP，就是国内生产总值，一个时间段内，一个国家所生产的所有产品和服务的总价值。它的计算方法通常是：消费，加政府采购，加净出口，加投资。
  • 乘数效应。你的消费，都是别人的收入。他的收入，又会变成新的消费，或者投资，GDP因此会越滚越大，这就是乘数效应。
    支持消费。从个人角度来看，无论是积攒到一定数量再消费，还是立刻消费，本质上都是要购买产品或服务，满足自己的欲望，无论是享受欲，炫耀欲还是求知欲。只不过是时间长短的区别罢了。从企业和国家角度来看，保持资金流动性是关键。只有钱运转起来，人与人的交易才能增多，才能使经济活跃。

微观经济学

供需理论

  • 在竞争市场中，供给和需求的相对稀缺性，决定商品的价格和产量。
  • 这种供需关系通过价格和竞争自我调节的现象，被亚当斯密称为看不见的手。
  • 中国大量的仿制品，价格之所以低廉，是供给极大增加，远超出有效需求，价格迅速下跌。
  • 很多中国企业亏损是因为供给侧太同质化，产能严重过剩。
  • 如何调节商业策略？
  • 提供稀缺的商品和内容。
  • 提供稀缺的能力和方法。
  • 没有被充沛供给的，都是稀缺。

  • 稀缺的不是知识，是加工、传递知识的能力。
